# 📝 Task Manager

A simple command-line Task Manager built with Python and PostgreSQL.

## Features

- Add task
- Show all tasks
- Count tasks
- Delete task
- Update task
- Complete task
- Search task by ID
- Show completed tasks

## Technologies

- Python
- PostgreSQL
- psycopg2
- Git
- GitHub

## How to run

1. Install PostgreSQL
2. Create your database
3. Update `database.py` with your own credentials
4. Run:

```bash
python main.py
```
